# Hai-Long Qin — Academic Homepage

Personal academic homepage of **Hai-Long Qin (秦海龙)**, Master Candidate at the
Beijing University of Posts and Telecommunications (BUPT). The site presents my
publications, research programs, internships, awards, and contact information.

## Tech stack

- React 18 + Vite 5
- Deployed to GitHub Pages via `npm run deploy`

## Local development

```bash
npm install
npm run dev      # start the local dev server
npm run build    # production build into dist/
npm run deploy   # publish dist/ to the gh-pages branch
```
